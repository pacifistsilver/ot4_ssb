## Getting Started
To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pacifistsilver/ot4_ssb.git
   ```
2. Activate Conda environment
   ```sh
   conda env create -f environment.yml
   ```
